# Proyecto: Ejercicios JavaScript - Tarea 1

Este repositorio contiene una serie de ejercicios de JavaScript que cubren conceptos fundamentales de programación web interactiva.

## Descripción
Repositorio para la entrega de la primera tarea del Módulo de Javascript del máster en desarrollo Fullstack de Conquer Blocks.

## Estructura del Proyecto

```
tarea1-js/
├── index.html          # Página principal
├── package.json        # Configuración del proyecto
├── public/             # Archivos públicos
├── src/                # Código fuente
│   ├── css/
│   │   └── styles.css  # Estilos CSS
│   ├── counter.js      # Contador básico
│   ├── index.html      # Página índice del src
│   └── ejercicio1/     # Ejercicio 1: [Cambiar color de fondo]
│   └── ejercicio2/     # Ejercicio 2: Contador de clics
│   └── ejercicio3/     # Ejercicio 3: Lista interactiva
│   └── ejercicio4/     # Ejercicio 4: Filtro de busqueda
│   └── ejercicio5/     # Ejercicio 5: Calculadora sencilla
│   └── ejercicio6/     # Ejercicio 6: Temporizador
│   └── ejercicio7/     # Ejercicio 7: Generador de contraseñas
│   └── ejercicio8/     # Ejercicio 8: Contador de caracteres y palabras
│   └── ejercicio9/     # Ejercicio 9: Lista de tareas con localStorage
```

## Ejercicios Incluidos

### Ejercicio 6: Temporizador con Inicio, Pausa y Reinicio
- Temporizador que cuenta horas, minutos y segundos
- Botones para iniciar, pausar y reiniciar
- Uso de `setInterval()` y `clearInterval()`

### Ejercicio 7: Generador de Contraseñas Aleatorias
- Genera contraseñas con longitud personalizable
- Incluye letras, números y caracteres especiales
- Validación de longitud mínima

### Ejercicio 8: Contador de Caracteres y Palabras
- Cuenta caracteres en tiempo real (sin espacios)
- Cuenta palabras separadas por espacios
- Actualización automática mientras escribes

### Ejercicio 9: Lista de Tareas con localStorage
- Agregar y eliminar tareas
- Marcar tareas como completadas
- Persistencia de datos con localStorage
- Limpiar tareas completadas

## Cómo usar

1. Clona el repositorio
2. Abre cualquier archivo HTML en tu navegador
3. Interactúa con los ejercicios

## Tecnologías

- HTML5
- CSS3
- JavaScript (ES6+)
- localStorage API
- DOM Manipulation
